# 鎮守府ぐらし！ - KanColleElectron - 
Electron製のMac用の艦これブラウザです。

## Features
* OS X El Capitan 10.11.6 で動作確認
* ゲーム画面のみのウィンドウ表示
* スクリーンショット機能
* 音量消音ON/OFF機能
* 常に最前面にウィンドウを置く機能
* スクリーンショットのTwitter投稿
